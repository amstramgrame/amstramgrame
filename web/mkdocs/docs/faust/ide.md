# Le Faust Web IDE

Une fois l'IDE ouvert, clique sur le bouton "Run" pour lancer le programme dans le navigateur Web :

<figure>
<img src="img/ide-run.svg" class="mx-auto d-block" width="100%">
</figure>

Un son devrait être produit ! Si tu ne l'entends pas, assures-toi que que le son de ton ordinateur n'est pas coupé. 

Toute modification faite dans le programme Faust sera reflétée automatiquement dans le son produit. Par exemple, essaye de changer la valeur du paramètre `freq` :

```
freq = 250;
```

Cliquer sur le bouton "Run" a également permis d'ouvrir une fenêtre qui correspond au programme Faust exécuté. Pour le stopper (et donc arrêter le son), clique sur la croix de fermeture de la fenêtre :

<figure>
<img src="img/ide-close.svg" class="mx-auto d-block" width="100%">
</figure>
